# 🛒 Amazon Sales Data Analysis using SQL
## 📌 Overview

This repository contains SQL queries used to perform Exploratory Data Analysis (EDA) on an Amazon sales dataset. The analysis focuses on understanding sales performance, customer behavior, product category profitability, and revenue trends.

Using SQL, the project extracts valuable insights from transactional data to support data-driven decision making.

### 🎯 Objectives

The main goals of this analysis are:

Explore the structure of the Amazon sales dataset

Calculate total sales, revenue, and profit

Analyze average price, revenue, and customer ratings

Identify top-performing product categories

Examine regional sales performance

Determine yearly revenue trends

Rank product categories based on revenue

### 🛠 Tools Used

SQL

MySQL


### 🔎 Analysis Performed

The SQL queries in this repository perform the following analyses:

##### 1️⃣ Dataset Exploration

Display all records from the dataset

Count the total number of rows in the dataset

#### 2️⃣ Product Category Analysis

Calculate total profit by product category

Identify top-performing categories

#### 3️⃣ Sales Performance Metrics

Average product price

Average total revenue

Average profit

Average customer rating

#### 4️⃣ Overall Business Performance

Total quantity sold

Total revenue generated

Total profit

#### 5️⃣ Time-Based Analysis

Analyze yearly revenue trends using order date

#### 6️⃣ Customer Behavior

Analyze payment method usage

Identify most common payment types

#### 7️⃣ Regional Sales Analysis

Calculate total revenue by customer region

Rank regions based on revenue using window functions

#### 8️⃣ Top Product Categories by Year

Use CTE and DENSE_RANK() to identify the top 3 product categories each year based on revenue.

### Key SQL Concepts Demonstrated

Aggregate Functions (SUM, AVG, COUNT)

GROUP BY and ORDER BY

Window Functions (DENSE_RANK)

Common Table Expressions (CTE)

Data Aggregation and Business Insights

### Conclusion

This project demonstrates how SQL can be used to perform efficient exploratory data analysis on e-commerce datasets. The queries help uncover important insights about sales trends, product performance, and regional revenue distribution.
