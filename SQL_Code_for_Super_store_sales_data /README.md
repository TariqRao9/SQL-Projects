# Super Store Sales Data Analysis (SQL)

This project focuses on performing Exploratory Data Analysis (EDA) on a Super Store Sales dataset using SQL. The objective is to extract meaningful insights, identify business trends, and support data-driven decision-making.

## 📊 Key Analysis Performed
- Total records and overall dataset overview
- Aggregation of key metrics:
  - Total Sales
  - Total Profit
  - Total Quantity Sold
  - Total Shipping Cost
  - Total Discount Given
- Sales analysis by:
  - Customer Segment
  - Individual Customers (Top Performers)
  - Product Sub-Categories
  - Country-wise performance
  - Yearly sales trends
- Identification of:
  - Top 5 customers using window functions (DENSE_RANK)
  - Top 3 performing countries per year

## 🛠️ Tools & Technologies
- SQL (MySQL / Relational Database)
- Window Functions
- Common Table Expressions (CTEs)
- Aggregate Functions

## 📈 Key Insights
- High-performing customer segments and regions
- Sales distribution across product categories
- Year-over-year business trends
- Top revenue-generating customers and markets

## 📁 Dataset
The dataset represents retail sales data including customer details, product categories, sales, profit, discounts, and shipping costs.

## 🚀 Purpose
This project demonstrates practical SQL skills for data analysis, including querying, aggregation, ranking, and business insight generation.
